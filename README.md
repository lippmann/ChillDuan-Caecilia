# 寒蝉端黑宋 Caecilia / ChillDuan Caecilia

基于 [寒蝉端黑宋 v1.2](https://github.com/Warren2060/ChillDuanHeiSong)（ChillDuanHeiSong）与 PMN Caecilia LT 67 Cond Medium（西文）合并而成的字体家族。中文部分保留寒蝉端黑宋原有字形，西文（ASCII U+0020–U+007E）替换为 Caecilia 的字形，实现中西文风格统一。

## 关于寒蝉端黑宋

基于思源宋体（SourceHanSerifSC）修改，横向加粗，字体横竖粗细相差更小，结合了黑体与宋体的特征，既有黑体粗壮的特性，又有宋体优雅的饰角，比黑体更雅致，比宋体更硬朗。介于黑体和宋体之间的基础字体风格。

原项目：[Warren2060/ChillDuanHeiSong](https://github.com/Warren2060/ChillDuanHeiSong)
